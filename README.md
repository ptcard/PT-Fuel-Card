# PT Fuel Card Application

A digital fuel card application form for Petro Trade Inc.

## 🔗 Live Form
[Open Application Form](https://ptcard.github.io/PT-Fuel-Card/)

## 📋 Features
- Personal information collection
- Vehicle / asset details with card limitations
- PT E-Portal access and notification settings
- Security questions
- Digital signature capture
- ID / Passport document upload
- Generates a downloadable PDF application

## 📄 Usage
Fill out all required fields marked with * and click **Download as PDF** to generate your application.

## 📬 Submit
Send the completed PDF to: sales@petrotrade.ws
